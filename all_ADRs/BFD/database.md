## Select database backend

Criteria:

* Mature / feature-ful.
* Good performance with known routes to scaling.
* Potential for Geo-data.
* Supported by Django.

Candidates:

* Sqlite 
* Postgresql
* MySql

Commentary:

Postgresql is the only solution that meets all the criteria.

---

* Status: approved.
* Decision: Postgresql.
* Author: ntoll.
